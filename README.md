# Local Development

The following assumes the use of `node@>=10`.

## Clone Repository

`git clone https://github.com/Uniswap/uniswap-v2-periphery.git`

## Install Dependencies

`yarn`

## Compile Contracts

`yarn compile`

## Run Tests

`yarn test`
