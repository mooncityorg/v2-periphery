## Local Development

The following assumes the use of `node@>=10`.

### Clone Repository

```
git clone https://github.com/Uniswap/uniswap-v2-periphery.git
cd uniswap-v2-periphery
```

### Install Dependencies

```
yarn
```

### Compile Contracts and Run Tests

```
yarn compile
yarn test
```
